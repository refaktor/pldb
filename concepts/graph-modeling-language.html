Graph Modeling Language (GML) is a hierarchical ASCII-based file format for describing graphs. It has been also named Graph Meta Language.. Read more on Wikipedia...

  • Graph Modeling Language Wikipedia page
  • There are at least 12,488 Graph Modeling Language repos on GitHub
  • Graph Modeling Language first developed in University of Passau
  • file extensions for Graph Modeling Language include gml
  • See also: (3 related languages)GraphML, ASCII, Python
  • Read more about Graph Modeling Language on the web: 1.
