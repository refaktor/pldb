DOT is a text markup language created in 2007.

#163on PLDB 16Years Old 28kRepos

DOT is a plain text graph description language. DOT graphs are typically files with the file extension gv or dot. The extension gv is preferred to avoid confusion with the extension dot used by early (pre-2007) versions of Microsoft Word. Read more on Wikipedia...

  • DOT Wikipedia page
  • DOT docs
  • There are at least 27,831 DOT repos on GitHub
  • DOT first developed in AT&T Labs Research
  • file extensions for DOT include dot and gv
  • The Google BigQuery Public Dataset GitHub snapshot shows 67 users using DOT in 68 repos on GitHub
  • ANTLR grammar for DOT
  • Pygments supports syntax highlighting for DOT
  • GitHub supports syntax highlighting for DOT
  • There is 1 Jupyter Kernel for DOT
