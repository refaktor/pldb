Fuzzy Markup Language, aka Fuzzy Markup Language, is a xml format created in 2004.

#627on PLDB 19Years Old

Fuzzy Markup Language (FML) is a specific purpose markup language based on XML, used for describing the structure and behavior of a fuzzy system independently of the hardware architecture devoted to host and run it.. Read more on Wikipedia...

  • Fuzzy Markup Language Wikipedia page
  • Fuzzy Markup Language first developed in University of Salerno
  • See also: (1 related languages)XML
